# Observables

The core primitive of RxJS: a lazy push collection of multiple values.

Topics that belong here: creating observables (`of`, `from`, `fromEvent`, `new Observable`), cold vs hot, lazy evaluation, the observable lifecycle, finite vs infinite streams.
